Simulation 2D World

Симуляция 2D мира, наполненная различными объектами:
    -Травоядные - передвигаются к ближайшей травы, потом едят ее
    -Хищники - передвигаются к ближайшему травоядному, атакует его или ест
    -Трава - является ресурсом для травоядных
    -Скала - является элементом ландшафта, препятствующим перемещению
    -Дерево - является элементом ландшафта, препятствующим перемещению

Хищиники и Травоядные перемещаются и поедают цель только по горизонтали и вертикали
